# MediCare Pharmacy Chain — Business Intelligence Dashboard

A Power BI business intelligence solution designed for MediCare Pharmacy Chain (PVT) Ltd, a 15-branch healthcare retail company in Colombo, Sri Lanka.

---

##  Dashboard Overview

Built to replace a fragmented manual spreadsheet system with a centralised, data-driven BI solution monitoring:

-  **Sales performance** across all 15 branches
-  **Inventory levels** and low-stock alerts
-  **Medicine expiry risk** monitoring
-  **Branch-by-branch KPI** comparisons
-  **Supplier performance** tracking

---

##  What Was Built

- Designed a full data science solution for a real-world pharmacy business problem
- Imported and cleaned pharmacy sales and inventory data using Power Query
- Built DAX measures for sales totals, profit margins, expiry tracking, and branch KPIs
- Created an interactive Power BI dashboard with slicers and drill-through filters
- Evaluated data science tools (Power BI, Python, Tableau, SQL, Excel) against business needs
- Covered GDPR compliance, Sri Lanka's PDPA, data security threats and mitigation strategies
- Made justified business recommendations based on dashboard findings

##  How to Use

1. Download `Imnah_Ifthikar_-_POWER_BI.pbix`
2. Open in **Power BI Desktop** (free download from Microsoft)
3. If prompted, relink the data source to `medicare_pharmacy_data.csv`
4. Use the slicers and filters to explore branch performance, inventory, and sales insights
